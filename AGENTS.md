# AGENTS.md - Legitimus GreenTech Website

## Projeto
Novo site institucional da **Legitimus GreenTech** — greentech especializada em sustentabilidade empresarial com tecnologia, método e inteligência de dados.

## Stack
- **Framework**: Next.js 15 (App Router, TypeScript)
- **Styling**: Tailwind CSS 4 + shadcn/ui
- **Animações**: Framer Motion + scroll-driven animations
- **Fontes**: Google Fonts (Cinzel, Montserrat, Inter)
- **Deploy**: Vercel
- **Domínio**: legitimus.greentech (Hostinger)

## Estrutura do Site
```
/ (Home) — Hero, showcase 4 produtos, clientes, CTA
/inventario-gee — Inventário de Emissões de GEE
/esg — Agenda ESG
/lixo-zero — Certificação Lixo Zero
/maquinas-equipamentos — Máquinas e Equipamentos
/contato — Formulário de contato
```
- **Área do Cliente**: Link externo → Plataforma Evoluto

### Insights SEO/GEO dos concorrentes
1. **Climoo** usa Schema JSON-LD completo (Organization, WebSite, WebPage, SoftwareApplication) — replicar
2. **WayCarbon** tem schema com Organization + social profiles + WebSite search — bom para GEO
3. **Sustain** foca em "menos planilhas, mais resultados" — CTA eficaz, porém fraca em storytelling visual
4. **Master Ambiental** usa posicionamento de consultoria clássica — oportunidade de diferenciação
5. **Nenhum concorrente** otimiza para AI bots (GPTBot, PerplexityBot) — vantagem competitiva
6. Prazo GHG Protocol Brasil 2026: 5 de junho — oportunidade de conteúdo urgente
7. **Patch.io** (referência do cliente): Hero full-viewport, statement bold único, scroll animations, trust bar com logos cedo, dual CTA path ("Fale conosco" + "Acesse plataforma")
8. **Gap competitivo**: Sustain é forte em features mas fraco em design; Patch é forte em design mas não localizado para Brasil. Oportunidade = design impactante (Patch) + conhecimento ESG local (Sustain)
